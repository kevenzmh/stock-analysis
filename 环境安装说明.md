# Stock Analysis 环境安装说明

## 方法一：使用 environment.yml 文件（推荐）

1. 确保已安装 Anaconda 或 Miniconda
2. 在项目根目录下运行以下命令：

```bash
# 创建并激活环境
conda env create -f environment.yml

# 激活环境
conda activate stock-analysis
```

## 方法二：使用 requirements.txt 文件

如果你更喜欢使用 pip 管理依赖：

```bash
# 创建新的conda环境
conda create -n stock-analysis python=3.8

# 激活环境
conda activate stock-analysis

# 安装依赖
pip install -r requirements.txt
```

## 验证安装

运行以下命令验证环境是否正确安装：

```bash
python check_environment.py
```

## 注意事项

- 本环境基于 Python 3.8
- 包含了股票分析所需的主要库：pandas, numpy, matplotlib, ta-lib, rqalpha, pytdx 等
- TA-Lib 库可能需要额外的系统依赖，如遇到安装问题，请参考项目中的 .whl 文件

## 环境更新

如果需要更新环境配置文件：

```bash
# 导出当前环境
conda env export -n stock-analysis > environment.yml

# 更新现有环境
conda env update -f environment.yml --prune
```
