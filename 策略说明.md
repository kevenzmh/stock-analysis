## ⚙️ 策略参数

### 当前设置

| 参数 | 数值 | 说明 |
|------|------|------|
| 止盈 | 15% | 盈利目标 |
| 止损 | 8% | 最大亏损 |
| 回撤止盈 | 12% | 从最高点回撤 |
| 持仓周期 | 1-6周 | 预期持股时间 |

### 修改参数

编辑 `CeLue.py` 文件：

```python
# 在卖策略函数中修改
固定止盈 = 0.15  # 改为 0.20 = 20%止盈
固定止损 = 0.08  # 改为 0.10 = 10%止损
回撤止盈 = 0.12  # 改为 0.10 = 10%回撤
```
